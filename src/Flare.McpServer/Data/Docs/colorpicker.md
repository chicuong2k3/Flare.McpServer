# FlareColorPicker

Color picker with presets, hex input, and RGB sliders.

```razor
<FlareColorPicker @bind-Value="_color" Label="Brand Color" />
```

## Key Parameters
- `Value` / `@bind-Value` - hex color string
- `Presets` - preset swatch colors
- `ShowInput` - hex text input
- `ShowSliders` - RGB sliders
