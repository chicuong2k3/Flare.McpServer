# FlareChip

Chip/tag for filtering, selection, or categories with optional icons and close.

```razor
<FlareChip Color="FlareColor.Primary" Closeable="true">React</FlareChip>
<FlareChip Variant="Variant.Outlined" Icon="person">User</FlareChip>
```

## Key Parameters
- `Color` - `FlareColor`
- `Variant` - Filled, Outlined, Soft
- `Icon` - leading icon
- `Closeable` - show remove button
- `Selected` - toggle selection state
