# FlareBadge

Badge for counts, status indicators, or labels.

```razor
<FlareBadge Color="FlareColor.Error">5</FlareBadge>
<FlareBadge Dot="true" Color="FlareColor.Success" />
```

## Key Parameters
- `Color` - `FlareColor` background
- `Dot` - compact dot without content
- `Position` - position on parent element
- `Visible` - show/hide
