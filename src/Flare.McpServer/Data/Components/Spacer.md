# FlareSpacer (Spacer)

**Category**: Layout
**Tags**: layout, spacer, flex, fill
**API Reference**: [https://jrfrigat.github.io/Flare/api/FlareSpacer.html](https://jrfrigat.github.io/Flare/api/FlareSpacer.html)
**Documentation**: [https://github.com/jrfrigat/Flare/blob/main/docs/en/ai-agents.md](https://github.com/jrfrigat/Flare/blob/main/docs/en/ai-agents.md)

## Description

Flex spacer that fills remaining space in a flex container.

## Examples

```razor
<FlareStack Direction="StackDirection.Horizontal">
    <FlareText>Start</FlareText>
    <FlareSpacer />
    <FlareText>End</FlareText>
</FlareStack>
```
