# FlareColorCustomizer (ColorCustomizer)

**Category**: Theming
**Tags**: theme, color, customizer, palette, mode
**API Reference**: [https://jrfrigat.github.io/Flare/api/FlareColorCustomizer.html](https://jrfrigat.github.io/Flare/api/FlareColorCustomizer.html)

## Description

UI widget to customize theme colors at runtime with palette and mode selection.

## Parameters

| Name | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `ShowPalettePicker` | `bool` | No | `true` | Show palette selector |
| `ShowModeToggle` | `bool` | No | `true` | Show mode toggle |

## Examples

```razor
<FlareColorCustomizer />
```
