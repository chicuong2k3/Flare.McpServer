# FlareScrollTop (ScrollTop)

**Category**: Navigation
**Tags**: scroll, top, navigation, button, floating
**API Reference**: [https://jrfrigat.github.io/Flare/api/FlareScrollTop.html](https://jrfrigat.github.io/Flare/api/FlareScrollTop.html)
**Documentation**: [https://github.com/jrfrigat/Flare/blob/main/docs/en/ai-agents.md](https://github.com/jrfrigat/Flare/blob/main/docs/en/ai-agents.md)

## Description

Floating button that appears when scrolling down to quickly jump to top.

## Parameters

| Name | Type | Required | Default | Description |
|------|------|----------|---------|-------------|
| `VisibleAfter` | `int` | No | `300` | Scroll px before showing |
| `Color` | `FlareColor` | No | `Default` | Button color |

## Examples

```razor
<FlareScrollTop />
```
